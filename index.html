The Code

Train Game 2015 is written in Java and powered by libGDX, a lightweight 2d game development library, so that developers spend less time writing boilerplate code, and more time implementing awesome gameplay.

Sample Code

All of our code has succint comments so that you always know how, and why things have been done a certain way, this makes our game easy to extend and modify.


    /**
     * Returns the Intersection at the given point.
     * @param point The point where an intersection lies.
     * @return The intersection at the point given, if it exists.
     * If it does not then null is returned.
     */
    public Intersection getIntersection(Point point) {
        if (intersections.isEmpty()) return null;

        for (Intersection intersection : intersections) {
            if (point.equals(intersection.getPoint())) return intersection;
        }

        return null;
    }
